# node-sass版本

因为node-sass版本过新(7.1.0),导致项目报错。这里推荐设置完下载源，再进行指定版本下载

npm config set sass_binary_site=https://npm.taobao.org/mirrors/node-sass

npm install node-sass@4.14.1 --save-dev

# ts版本
